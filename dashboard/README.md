# Financial Inclusion Dashboard

## Overview
This Streamlit dashboard allows stakeholders to explore financial inclusion indicators,
understand historical trends, and view scenario-based forecasts.

## Features
- Key metrics overview
- Interactive trend analysis
- Scenario-based forecasts
- Inclusion target tracking (60%)

## How to Run Locally

1. Create and activate virtual environment
2. Install dependencies:
   pip install streamlit pandas plotly
3. Navigate to dashboard folder
4. Run:
   streamlit run app.py

## Data
All data is stored in `data/processed/`
